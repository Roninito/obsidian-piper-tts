# Changelog

## [1.0.0] - 2025-02-25

### Initial Release

- Text-to-speech using local Piper TTS engine
- Voice browser with 200+ voices: search, filter, preview, download
- Commands: read selection, read note, pause/resume, stop, export to WAV
- Settings: configurable piper binary path, models directory, speed, chunk size
- Status bar playback indicator
- Markdown stripping before synthesis
- Sentence-level chunking for low-latency playback on long notes
